  ---
  对 E1 nsys/ncu profiling 的关键提示

  1. nsys 采样时：--trace cuda,nvtx,cublas 即可覆盖所有 kernel。FA2 的 kernel 以 flash_attention 前缀出现，VisiPruner pruning kernel 以 cosine_similarity/norm/where/topk 前缀出现。
  2. ncu 分析时：VisiPrunerLlamaAttention 的 kernel 名称来自 PyTorch 的 CUDA kernel launch，不是源码函数名。nsys 的 cuda_gpu_kern_sum 报告会给出实际的 CUDA kernel 名（如 void
  at::native::cosine_similarity_kernel<...>），用那些 kernel 名填入 ncu 的 --kernel-name 参数。
  3. NVTX marker 插入点：如需分阶段 nsys profiling（E3/E4 需要），NVTX range 应插入在 custom_modeling_llama.py:605（VisiPrunerLlamaAttention.forward）的各算子边界处，以及
  custom_modeling_llama.py:1365（layer loop）的 prefill/decode 分支处。
